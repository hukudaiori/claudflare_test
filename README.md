# Claudflare Pages Sample

Claudflare Pages用のサンプルホームページです。このリポジトリをベースにしてホームページをカスタマイズできます。

## ファイル構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `script.js` - JavaScriptファイル
- `README.md` - このファイル

## セットアップ方法

1. このリポジトリをClone
2. ローカルで確認（`index.html`をブラウザで開く）
3. 内容をカスタマイズ
4. Claudflare Pagesにデプロイ

## Claudflare Pagesでのデプロイ

1. Claudflare Dashboardにログイン
2. Pages > Create a project > Connect to Git
3. このリポジトリを選択
4. Build settingsをデフォルトの��ま使用
5. デプロイ完了

## カスタマイズ

- `index.html` - ページコンテンツを編集
- `styles.css` - デザインをカスタマイズ
- `script.js` - 機能を追加
